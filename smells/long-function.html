Funções Longas

Long Function

📖 Definição

Uma Função Longa é um método que contém muitas linhas de código e tenta realizar múltiplas tarefas simultaneamente. Geralmente, quando você precisa rolar a tela várias vezes para ver todo o método, é um sinal claro de que ele está muito longo.

Funções longas são difíceis de entender, testar e manter. Elas violam o princípio da Responsabilidade Única (SRP) e tornam o código propenso a bugs.

⚠️ Problemas Causados

  • Dificuldade de compreensão: Muito código em um só lugar torna difícil entender a lógica.
  • Testabilidade reduzida: Testar uma função que faz muitas coisas requer muitos casos de teste.
  • Reutilização limitada: Código embutido em funções longas não pode ser reutilizado facilmente.
  • Manutenção complexa: Mudanças podem ter efeitos colaterais inesperados.
  • Debug difícil: Encontrar bugs em funções longas é como procurar agulha no palheiro.
  • Violação do SRP: A função tem mais de um motivo para mudar.

🔍 Como Identificar

  • Método com mais de 20-30 linhas de código.
  • Você precisa rolar a tela para ver todo o método.
  • A função contém múltiplos blocos lógicos separados por comentários.
  • Nomes de variáveis genéricos como temp, data, result.
  • Múltiplos níveis de indentação (if/for/while aninhados).
  • A função faz coisas que poderiam ter nomes próprios descritivos.

❌ Java - Função Longa 
public class ProcessadorPedidos {
    
    public void processarPedido(Pedido pedido) {
        // Validar dados do pedido
        if (pedido == null) {
            throw new IllegalArgumentException("Pedido não pode ser nulo");
        }
        if (pedido.getItens() == null || pedido.getItens().isEmpty()) {
            throw new IllegalArgumentException("Pedido deve ter itens");
        }
        if (pedido.getCliente() == null) {
            throw new IllegalArgumentException("Cliente não informado");
        }
        
        // Calcular subtotal
        double subtotal = 0;
        for (Item item : pedido.getItens()) {
            subtotal += item.getPreco() * item.getQuantidade();
        }
        
        // Calcular desconto
        double desconto = 0;
        if (pedido.getCliente().isVip()) {
            desconto = subtotal * 0.15;
        } else if (subtotal > 1000) {
            desconto = subtotal * 0.10;
        } else if (subtotal > 500) {
            desconto = subtotal * 0.05;
        }
        
        // Calcular frete
        double frete = 0;
        String estado = pedido.getCliente().getEstado();
        if (estado.equals("SP") || estado.equals("RJ")) {
            frete = 15.0;
        } else if (estado.equals("MG") || estado.equals("ES")) {
            frete = 20.0;
        } else {
            frete = 30.0;
        }
        
        // Calcular total
        double total = subtotal - desconto + frete;
        
        // Aplicar imposto
        double imposto = total * 0.18;
        total += imposto;
        
        // Salvar no banco
        PedidoRepository repo = new PedidoRepository();
        pedido.setSubtotal(subtotal);
        pedido.setDesconto(desconto);
        pedido.setFrete(frete);
        pedido.setTotal(total);
        repo.salvar(pedido);
        
        // Enviar email de confirmação
        EmailService emailService = new EmailService();
        emailService.enviarConfirmacao(pedido.getCliente().getEmail(), pedido);
        
        // Atualizar estoque
        for (Item item : pedido.getItens()) {
            EstoqueService estoque = new EstoqueService();
            estoque.reservar(item.getProdutoId(), item.getQuantidade());
        }
        
        // Log da operação
        System.out.println("Pedido " + pedido.getId() + " processado com sucesso!");
    }
}

✅ Java - Funções Extraídas 
public class ProcessadorPedidos {
    
    public void processarPedido(Pedido pedido) {
        validarPedido(pedido);
        
        double subtotal = calcularSubtotal(pedido);
        double desconto = calcularDesconto(pedido, subtotal);
        double frete = calcularFrete(pedido);
        double total = calcularTotal(subtotal, desconto, frete);
        
        pedido.setSubtotal(subtotal);
        pedido.setDesconto(desconto);
        pedido.setFrete(frete);
        pedido.setTotal(total);
        
        salvarPedido(pedido);
        enviarConfirmacao(pedido);
        atualizarEstoque(pedido);
        
        logSucesso(pedido);
    }
    
    private void validarPedido(Pedido pedido) {
        if (pedido == null) {
            throw new IllegalArgumentException("Pedido não pode ser nulo");
        }
        if (pedido.getItens() == null || pedido.getItens().isEmpty()) {
            throw new IllegalArgumentException("Pedido deve ter itens");
        }
        if (pedido.getCliente() == null) {
            throw new IllegalArgumentException("Cliente não informado");
        }
    }
    
    private double calcularSubtotal(Pedido pedido) {
        return pedido.getItens().stream()
            .mapToDouble(i -> i.getPreco() * i.getQuantidade())
            .sum();
    }
    
    private double calcularDesconto(Pedido pedido, double subtotal) {
        if (pedido.getCliente().isVip()) {
            return subtotal * 0.15;
        } else if (subtotal > 1000) {
            return subtotal * 0.10;
        } else if (subtotal > 500) {
            return subtotal * 0.05;
        }
        return 0;
    }
    
    private double calcularFrete(Pedido pedido) {
        String estado = pedido.getCliente().getEstado();
        return switch (estado) {
            case "SP", "RJ" -> 15.0;
            case "MG", "ES" -> 20.0;
            default -> 30.0;
        };
    }
    
    private double calcularTotal(double subtotal, double desconto, double frete) {
        double totalParcial = subtotal - desconto + frete;
        return totalParcial + (totalParcial * 0.18);
    }
    
    private void salvarPedido(Pedido pedido) {
        new PedidoRepository().salvar(pedido);
    }
    
    private void enviarConfirmacao(Pedido pedido) {
        new EmailService().enviarConfirmacao(
            pedido.getCliente().getEmail(), 
            pedido
        );
    }
    
    private void atualizarEstoque(Pedido pedido) {
        EstoqueService estoque = new EstoqueService();
        for (Item item : pedido.getItens()) {
            estoque.reservar(item.getProdutoId(), item.getQuantidade());
        }
    }
    
    private void logSucesso(Pedido pedido) {
        System.out.println("Pedido " + pedido.getId() + " processado com sucesso!");
    }
}

🔧 Técnicas de Refatoração

Extract Method (Extrair Método)

Identifique blocos de código que realizam uma tarefa específica e extraia-os para métodos separados com nomes descritivos. Esta é a técnica principal para lidar com funções longas.

Replace Temp with Query (Substituir Temporária por Query)

Substitua variáveis temporárias por métodos que calculam o valor quando necessário, reduzindo a complexidade do método principal.

Decompose Conditional (Decompor Condicional)

Extraia condições complexas e seus blocos correspondentes para métodos separados, tornando a lógica mais clara.

Replace Loop with Pipeline (Substituir Loop por Pipeline)

Substitua loops complexos por operações de stream/lambda mais expressivas e concisas.
